# Privacy Policy for Hero Wars Snapshot

Last updated: July 14, 2026

Hero Wars Snapshot processes gameplay and account-progression information only
to provide its user-requested snapshot, viewer, and JSON export features.

## Data processed

The extension processes recognized Hero Wars responses containing heroes,
titans, pets, inventory, teams, resources, and related progression data. It
also reads public game localization and asset indexes from allowlisted game CDN
hosts to display names and images.

The extension is designed to discard request bodies, authentication context,
cookies, headers, passwords, email addresses, chat, billing information, and
unrecognized network responses.

## Storage and sharing

Sanitized snapshots and resolved public image URLs are stored locally using
Chrome extension storage on the user's device. Game Tool Works does not
collect, receive, sell, transmit, or share this data. Removing the extension
through Chrome removes its extension storage.

The user may explicitly download a snapshot as a JSON file. That downloaded
file is controlled by the user and is not uploaded by the extension.

## External links

The popup contains an optional support link to Buy Me a Coffee. It opens only
after an explicit click. No snapshot data or identifiers are appended to that
link. Buy Me a Coffee applies its own privacy policy to visits and payments.

## Retention and deletion

The most recent snapshot remains in local extension storage until it is
replaced, the extension's storage is cleared, or the extension is removed.

## Contact

Feedback and privacy questions may be submitted through the public GitHub
repository maintained by Game Tool Works.

Hero Wars Snapshot's use of information received from Chrome APIs adheres to
the Chrome Web Store User Data Policy, including its Limited Use requirements.
